2.补充几点
常用快捷键：
Alt+Entry：换行
Alt+F11：打开程序窗口
F5：打开运行宏窗口

函数：
Mid(String,start[,Length])
String - 必需的参数。输入从中返回指定数量的字符的字符串。
Start - 必需的参数。 一个整数，它指定了字符串的起始位置。
Length - 必需的参数。 一个整数，指定要返回的字符数。
注意：Start需要从1开始。
